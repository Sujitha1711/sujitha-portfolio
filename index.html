In this Hackathon, our team developed a website to automate test case execution for banking applications. The platform eliminates the need for manual intervention by automating test scenarios using Jenkins.

We designed a user-friendly interface and conducted Unit and UI Testing. Our testing also included cross-browser checks across Chrome, Edge, and Firefox.
